## Icon 标题

### 使用指南
``` javascript
import { Icon } from 'icui';

Vue.component(Icon.name, Icon);
```

### 代码演示

#### 

:::demo
```html
<ic-icon name="del"></ic-icon>
```
:::

### Options

| 参数       | 说明      | 类型       | 默认值       |
|-----------|-----------|-----------|-------------|
| name | icon的name | `String` | ``|
