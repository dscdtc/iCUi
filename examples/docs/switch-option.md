## Switch 滑动开关

### 使用指南
``` javascript
import { SwitchOption } from 'icui';

Vue.component(SwitchOption.name, SwitchOption);
```

### 代码演示

#### 

:::demo
```html
<ic-switch :value="false" :name="name"></ic-switch>
```
:::

### Options

| 参数       | 说明      | 类型       | 默认值       |
|-----------|-----------|-----------|-------------|
| value | 开关状态 | `Boolean` | false|
| name | switch 的 name | `String` | ''|
